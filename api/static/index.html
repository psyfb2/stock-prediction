This tool will give you a bullish rating for a stock as a percentage. A rating of 0% would be very bearish and 100% would be very bullish. Concretely, the percentage represents the probability that a Take-Profit of 6% is hit before a Trailing-Stop-Loss of 4% is hit, relative to the current days open price. For more information checkout the github README for this project here.

Warning: the rating is calculated using technical analysis and does not take fundementals into account. Therefore, take the rating with a grain of salt, especially if there is an upcoming event (e.g. earnings, news, etc). Furthermore, the developer of this tool cannot be held liable for any loss of capital, use at your own risk!
